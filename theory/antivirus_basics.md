# Antivirus Basics

Questo documento fornisce una panoramica delle nozioni fondamentali sugli antivirus:
cosa sono, come funzionano e quali componenti li rendono efficaci nella protezione dei sistemi.

## Cos'è un antivirus

Un antivirus è un software di sicurezza progettato per:
- Rilevare
- Prevenire
- Bloccare
- Rimuovere

software malevolo (*malware*) come virus, worm, trojan, ransomware, spyware e altre minacce.

## Obiettivi principali di un antivirus

- Proteggere il sistema operativo
- Salvaguardare i dati dell'utente
- Prevenire accessi non autorizzati
- Ridurre il rischio di infezioni future

## Componenti principali di un antivirus

### Motore di scansione
Analizza file, memoria e processi alla ricerca di comportamenti o firme sospette.

### Database delle firme
Contiene le definizioni dei malware conosciuti.
Deve essere aggiornato frequentemente.

### Modulo di protezione in tempo reale
Monitora costantemente:
- File aperti o eseguiti
- Download
- Processi attivi

### Sistema di quarantena
Isola i file sospetti per impedirne l'esecuzione.

### Modulo di aggiornamento
Mantiene il software e le firme sempre aggiornate.

## Tipi di scansione

- **Scansione completa**: analizza l'intero sistema
- **Scansione rapida**: controlla le aree critiche
- **Scansione personalizzata**: scelta manuale dei percorsi
- **Scansione in tempo reale**: continua e automatica

## Importanza degli aggiornamenti

Un antivirus non aggiornato è inefficace.
Le nuove minacce emergono continuamente e richiedono:
- Nuove firme
- Miglioramenti agli algoritmi di rilevamento

## Limitazioni degli antivirus

- Non garantiscono protezione al 100%
- Possono generare falsi positivi
- Possono essere aggirati da malware avanzati
- Dipendono dal comportamento dell'utente

## Buone pratiche complementari

- Mantenere il sistema operativo aggiornato
- Non eseguire file sospetti
- Usare firewall
- Fare backup regolari
- Usare account con privilegi limitati

## Conclusione

Gli antivirus rappresentano un elemento fondamentale della sicurezza informatica,
ma devono essere considerati parte di una strategia di difesa più ampia.
